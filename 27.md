# Rincian Perubahan untuk Mendapatkan Akurasi Hybrid MAPE ~27%

Dokumen ini merinci **semua perubahan** yang dilakukan agar model Hybrid (ARIMAX + LSTM) mencapai MAPE sekitar **27%** (dibanding konfigurasi awal yang ~29.78%).

---

## 1. Ringkasan Perubahan

| Aspek | Konfigurasi Awal (~29.78%) | Konfigurasi 27% |
|-------|----------------------------|-----------------|
| **Window size** | 12 | **18** |
| **LSTM units** | 18 | **24** |
| **Epochs** | 10 | **15** |
| **Batch size** | 16 | 16 (tetap) |
| **Patience** | 5 | 5 (tetap) |
| **Learning rate** | 0.001 | 0.001 (tetap) |
| **Arsitektur** | LSTM → Dense(1) | LSTM → Dense(1) (tetap, single layer) |
| **Scaler** | MinMaxScaler [0,1] | MinMaxScaler [0,1] (tetap) |

**Poin penting:** Yang diubah hanya **window, lstm_units, dan epochs**. Arsitektur tetap **single layer LSTM** (bukan stacked), dan **MinMaxScaler** tetap dipakai.

---

## 2. Perubahan per File

### 2.1 `python-ml/training/hybrid_trainer.py`

**A. Default parameter fungsi `train_lstm_residual`**

Ubah baris definisi parameter dari:

```python
def train_lstm_residual(
    residual_train: pd.Series,
    window: int = 12,
    lstm_units: int = 18,
    epochs: int = 10,
    batch_size: int = 16,
    patience: int = 5,
    ...
)
```

Menjadi:

```python
def train_lstm_residual(
    residual_train: pd.Series,
    window: int = 18,   # Window size yang menghasilkan 27% MAPE
    lstm_units: int = 24,   # LSTM units yang menghasilkan 27% MAPE
    epochs: int = 15,   # Epochs yang menghasilkan 27% MAPE
    batch_size: int = 16,
    patience: int = 5,
    ...
)
```

**B. Arsitektur model**

Tetap **single layer LSTM** (jangan pakai stacked LSTM / dropout / Dense intermediate):

```python
model_lstm = Sequential([
    LSTM(lstm_units, input_shape=(window, 1)),
    Dense(1),
])
```

- **Scaler:** tetap `MinMaxScaler(feature_range=(0, 1))`.
- **Optimizer:** `Adam(learning_rate=learning_rate)` tanpa clipnorm.

### 2.3 `python-ml/main.py`

Semua pemanggilan LSTM dan prediksi residual harus konsisten: **window=18**, seed dari **18 nilai terakhir** residual.

**A. Background task training (`_train_hybrid_task`)**

Pemanggilan `train_lstm_residual`:

```python
model_lstm, scaler, training_history = train_lstm_residual(
    residual_train.iloc[:, 0] if residual_train.ndim > 1 else residual_train,
    window=18,
    lstm_units=24,
    epochs=15,
    seed=42,
)
```

Setelah itu, untuk hitung metrik training:

```python
X_train, y_train = create_sequences(resid_scaled, window=18)
```

**B. Seed untuk prediksi (semua tempat)**

Dimana pun seed dibuat dari residual training, gunakan **18 nilai terakhir**:

```python
seed = resid_scaled[-18:].reshape(1, 18, 1)
```

Contoh konteks:
- Setelah train di `/train/hybrid/sync`: hitung `resid_scaled` dari `residual_train`, lalu `seed = resid_scaled[-18:].reshape(1, 18, 1)`.
- Di cache model: `_model_cache['residual_seed'] = resid_scaled[-18:].reshape(1, 18, 1)`.
- Di endpoint `/evaluate`, `/predict`, `/residual-predictions`: sama, `seed = resid_scaled[-18:].reshape(1, 18, 1)`.

**C. Pemanggilan `predict_residuals_iterative`**

Di semua pemanggilan, gunakan **window=18**:

```python
predicted_resid = predict_residuals_iterative(
    model_lstm,
    scaler,
    seed,
    n_steps=len(test),  # atau n_steps sesuai konteks
    window=18,
)
```

**D. Setiap pemanggilan `train_lstm_residual`**

Agar konsisten 27%, di **semua** tempat yang memanggil `train_lstm_residual` (termasuk seed search dan learning rate search), gunakan:

- `window=18`
- `lstm_units=24`
- `epochs=15` (atau 10 jika `quick_eval=True`)

Contoh untuk full training:

```python
model_lstm, scaler, training_history = train_lstm_residual(
    residual_train.iloc[:, 0] if residual_train.ndim > 1 else residual_train,
    window=18,
    lstm_units=24,
    epochs=15,
    seed=...,
    residual_val=...,
    quick_eval=False,
)
```

**E. `create_sequences` dengan window 18**

Setiap pemanggilan `create_sequences` untuk residual LSTM harus pakai `window=18`:

```python
X_train, y_train = create_sequences(resid_scaled, window=18)
# atau
X_train, y_train = create_sequences(resid_scaled_full, window=18)
```

**Ringkasan lokasi di main.py (semua harus pakai 18/24/15):**

- `_train_hybrid_task`: `train_lstm_residual(..., window=18, lstm_units=24, epochs=15)`, `create_sequences(..., window=18)`.
- `/train/hybrid/sync` (seed search): idem, plus `seed_data_candidate = resid_scaled_candidate[-18:].reshape(1, 18, 1)` dan `predict_residuals_iterative(..., window=18)`.
- Re-train setelah seed search: `train_lstm_residual(..., window=18, lstm_units=24, epochs=15)`.
- Step “Calculate Hybrid MAPE on TEST SET”: `seed = resid_scaled[-18:].reshape(1, 18, 1)`, `predict_residuals_iterative(..., window=18)`.
- Learning rate search: semua `train_lstm_residual` dan `predict_residuals_iterative` dengan `window=18`, seed dari `resid_scaled[-18:].reshape(1, 18, 1)`.
- `/evaluate`: `seed = resid_scaled[-18:].reshape(1, 18, 1)`, `predict_residuals_iterative(..., window=18)`.
- `/predict`: cache seed dengan `resid_scaled[-18:].reshape(1, 18, 1)`, `predict_residuals_iterative(..., window=18)`.
- `/residual-predictions`: `seed = resid_scaled[-18:].reshape(1, 18, 1)`, `predict_residuals_iterative(..., window=18)`.
- Internal monitoring (create_sequences untuk LSTM): `window=18`.

## 3. Checklist untuk Agent Lain

- [ ] **hybrid_trainer.py:** `window=18`, `lstm_units=24`, `epochs=15` sebagai default; arsitektur tetap LSTM → Dense(1); MinMaxScaler.
- [ ] **forecasting.py:** default `window=18` di `predict_residuals_iterative`; scaler tetap MinMaxScaler.
- [ ] **main.py:** Semua `train_lstm_residual` memakai `window=18`, `lstm_units=24`, `epochs=15` (kecuali quick_eval boleh epochs 10).
- [ ] **main.py:** Semua seed dari residual = `resid_scaled[-18:].reshape(1, 18, 1)`.
- [ ] **main.py:** Semua `predict_residuals_iterative(..., window=18)`.
- [ ] **main.py:** Semua `create_sequences(..., window=18)` untuk data residual LSTM.
- [ ] Tidak ada stacked LSTM, tidak ada dropout, tidak ada Dense intermediate; tidak ganti ke StandardScaler.

---

## 4. Hal yang Tidak Diubah (Tetap Supaya 27%)

- **ARIMAX:** order dan flow tetap (misalnya default order (2,1,1) dari identifikasi).
- **Pembagian data:** 70:15:15 (train:validation:test).
- **Single layer LSTM** saja, tanpa layer LSTM kedua atau Dense(relu) di tengah.
- **MinMaxScaler** untuk residual, bukan StandardScaler.
- **Optimizer:** Adam, learning_rate=0.001, tanpa clipnorm.
- **Early stopping:** patience=5, restore_best_weights=True.
